# MechaChaotic Site Design Specification

## Color Scheme (from original site)
- **Background:** #1a0c0c (dark brown/black)
- **Primary Accent:** #FF4500 (orange-red) 
- **Secondary Dark:** #0a0a0a (near black)
- **Text:** #f5f5f5 (off-white)
- **Border:** rgba(255, 69, 0, 0.3) (orange with transparency)

## Typography
- **Primary:** Montserrat (sans-serif)
- **Display:** Changa One (headers/titles)
- Weights: 100-900 for Montserrat

## Layout Structure
1. Hero section with large orange logo
2. Navigation with links
3. "Collaborate with Chaos" intro section
4. Carousel/slider with mecha images
5. Horizontal scrolling "Symbols of Power" section (5 cards)
6. Team section
7. FAQ section
8. Footer

## Key Visual Elements
- Orange MC logo prominently displayed
- Pattern overlays (Pattern A Red, Pattern B2, Pattern C)
- Mecha character images in carousel
- 5 Symbol of Power SVG icons (Dominance, Chaos, Apex, Corrupted, Beast)
- Dark background with orange accents throughout
